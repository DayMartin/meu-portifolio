Automação Jenkins e Teste SonarQube

Implantei uma arquitetura de automação com uma pipeline no Jenkins com acesso ao SonarQube para análise de código.

Apartir do mommento que o dev faz um commit no código de produção, o Jenkins implantado na EC2 identifica o commit, envia o código para o SonarQube analisar

Após a análise o Jenkins sobe uma image baseado no código, derruba os containeres em execução e sobre novos containeres baseado na nova imagem gerada

Quando o container está em execução é enviado para o CloudWatch logs para acompanhamento do código

Essa arquitetura foi projetada e implantada na empresa em que trabalho atualmente.
